Predictive Analysis of Groundwater Level Using Satellite Variables

Project Overview

Developed a machine learning model to predict groundwater levels using satellite data in Sri Lanka's Malwathu Oya Basin, demonstrating the effectiveness of remote sensing for water resource management.

Figure 1: Study Area - Malwathu Oya Basin in Sri Lanka

Key Contributions

  • Designed and implemented a comprehensive data collection system integrating satellite and ground-based measurements
  • Developed predictive models using multiple machine learning approaches
  • Demonstrated superior performance of ensemble methods over traditional regression
  • Identified key satellite-derived variables correlated with groundwater levels

Technical Skills Applied

Data Science

  • Machine Learning (Random Forest, Gradient Boosting)
  • Statistical Analysis (Correlation, Regression)
  • Feature Engineering

Remote Sensing

  • Landsat 8 Data Processing
  • NDVI and LST Extraction
  • Spatial Analysis

Methodology

Figure 2a: Research Methodology - Data Collection & Processing

Figure 2b: Research Methodology - Modeling & Analysis

  1. Data Collection

    Integrated groundwater measurements, rainfall data, and Landsat 8 imagery from multiple sources

  2. Feature Extraction

    Processed satellite data to derive key variables: Land Surface Temperature (LST) and Normalized Difference Vegetation Index (NDVI)

  3. Correlation Analysis

    Identified relationships between satellite variables and groundwater levels across different wells

    Figure 3: Correlation between Land Surface Temperature and Groundwater Levels

    Figure 4: Correlation between NDVI and Groundwater Levels

  4. Model Development

    Built and compared three prediction models: Linear Regression, Random Forest, and Gradient Boosting

  5. Performance Evaluation

    Assessed models using R² and MSE metrics, with ensemble methods showing superior performance

    Figure 5: Comparison of Model Performance (Actual vs Predicted Values)

Key Achievements

  • Achieved 76% prediction accuracy using Gradient Boosting model
  • Demonstrated strong correlations between LST and groundwater levels (up to r=0.72)
  • Proved the effectiveness of satellite methods for groundwater monitoring
